### 5.1 接口规范

智惠X4的API设计遵循RESTful风格，采用JSON格式进行数据交换。接口规范包括以下要点：

**基础URL**：生产环境为 `https://api.zhx4.com`，开发环境为 `https://api-dev.zhx4.com`

**请求方法**：
- GET：获取资源
- POST：创建资源
- PUT：更新资源
- DELETE：删除资源

**请求头**：
```
Content-Type: application/json
Authorization: Bearer {token}
X-Client-Version: 1.0.0
X-Platform: weapp
X-Device-ID: {deviceId}
```

**响应格式**：
```json
{
  "code": 0,
  "message": "success",
  "data": { },
  "timestamp": 1640000000
}
```

**错误码规范**：
- 0：成功
- 1000-1999：客户端错误（参数错误、权限不足等）
- 2000-2999：服务端错误（数据库错误、缓存错误等）
- 3000-3999：业务错误（余额不足、冷却中、对手不存在等）

### 5.2 签名验证

为了保障接口安全性，所有请求都需要进行签名验证：

```javascript
// 签名算法
// 1. 将请求参数按key字典序排序
// 2. 拼接成 key1=value1&key2=value2 格式
// 3. 在末尾拼接 secret_key
// 4. 对整个字符串进行 MD5/SHA256 加密
// 5. 生成签名 sign

function generateSign(params, secretKey) {
  // 排序
  const sortedKeys = Object.keys(params).sort();
  
  // 拼接
  let signString = '';
  for (const key of sortedKeys) {
    if (params[key] !== undefined && params[key] !== null) {
      signString += `${key}=${params[key]}&`;
    }
  }
  
  // 去掉末尾的 &
  signString = signString.slice(0, -1);
  
  // 拼接密钥
  signString += secretKey;
  
  // MD5加密
  return crypto.createHash('md5').update(signString).digest('hex');
}

// 验证签名
function verifySign(params, sign, secretKey) {
  const expectedSign = generateSign(params, secretKey);
  return sign === expectedSign;
}
```

    console.error('Error:', err);
    
    if (err.code >= 1000) {
      // 已知业务错误
      ctx.body = {
        code: err.code,
        message: err.message,
        data: null,
        timestamp: Date.now()
      };
    } else {
      // 未知错误
      ctx.body = {
        code: 2000,
        message: '服务器内部错误',
        data: null,
        timestamp: Date.now()
      };
    }
  }
}

### 6.2 组件设计

前端组件采用自底向上的设计方式，从基础组件到业务组件逐层构建。

**地图组件（Map）** 是LBS功能的核心组件，负责地图展示和交互。组件属性包括：centerPosition（中心位置）、scale（缩放级别）、markers（标记点数组）、circles（圆形区域）、showLocation（是否显示定位点）。组件方法包括：getCenterPosition（获取中心位置）、moveToLocation（移动到指定位置）、includeMarkers（缩放包含所有标记点）。

**战斗组件（Battle）** 负责对战流程的UI展示。子组件包括：BattleCard（对战卡片，显示双方信息）、BattleTimer（倒计时）、BattleEffect（战斗特效）、BattleResult（结果展示）。战斗状态管理包括：准备、进行中、结算三个阶段。

**道具组件（Item）** 负责道具的展示和使用。ItemIcon：道具图标组件，支持数量显示和冷却动画；ItemList：道具列表组件，支持滑动选择；ItemDetail：道具详情弹窗。

**宝箱组件（Chest）** 负责宝箱的展示和开启动画。组件状态包括：未开启、开启中、已开启三种状态，支持开启动画和奖励展示。

### 6.3 状态管理

微信小程序的状态管理可以采用多种方案，根据复杂度选择。

**方案一：小程序原生setData（推荐简单场景）**

适用于页面级状态管理，数据量较小的场景。

```javascript
// pages/index/index.js
Page({
  data: {
    userInfo: null,
    location: null,
    nearbyChests: [],
    markers: [],
    scale: 16,
    latitude: 30.5728,
    longitude: 104.0668
  },
  
  onLoad() {
    this.initLocation();
    this.initSocket();
  },
  
  onShow() {
    if (this.data.needRefresh) {
      this.fetchData();
    }
  },
  
  onUnload() {
    clearInterval(this.locationTimer);
    if (this.socket) {
      this.socket.close();
    }
  },
  
  initLocation() {
    wx.getLocation({
      type: 'gcj02',
      isHighAccuracy: true,
      success: (res) => {
        this.setData({
          latitude: res.latitude,
          longitude: res.longitude,
          location: res
        });
        this.loadNearbyElements(res);
        this.startLocationWatch();
      },
      fail: () => {
        wx.showToast({ title: '定位失败', icon: 'none' });
      }
    });
  },
  
  startLocationWatch() {
    wx.startLocationUpdate({
      success: () => {
        wx.onLocationChange(this.onLocationChange.bind(this));
      }
    });
  },
  
  onLocationChange(res) {
    const now = Date.now();
    if (now - (this.lastLocationUpdate || 0) < 2000) return;
    this.lastLocationUpdate = now;
    
    this.setData({
      latitude: res.latitude,
      longitude: res.longitude,
      location: res
    });
    
    this.syncLocation(res);
  },
  
  syncLocation(location) {
    const api = require('../../utils/api');
    api.updatePosition({
      latitude: location.latitude,
      longitude: location.longitude,
      accuracy: location.accuracy || 0,
      speed: location.speed || 0
    }).catch(err => {
      console.error('位置同步失败', err);
    });
  },
  
  loadNearbyElements(location) {
    const api = require('../../utils/api');
    api.getNearbyElements({
      latitude: location.latitude,
      longitude: location.longitude,
      radius: 500
    }).then(res => {
      this.setData({
        nearbyChests: res.data.chests,
        nearbyPlayers: res.data.players
      });
      this.updateMarkers();
    });
  },
  
  updateMarkers() {
    const markers = [];
    
    this.data.nearbyChests.forEach((chest) => {
      markers.push({
        id: `chest_${chest.id}`,
        latitude: chest.latitude,
        longitude: chest.longitude,
        iconPath: chest.type === 'rare' 
          ? '/images/map/chest-rare.png' 
          : '/images/map/chest-common.png',
        width: 40,
        height: 40,
        callout: {
          content: chest.name,
          color: '#FFFFFF',
          fontSize: 12,
          borderRadius: 4,
          bgColor: chest.type === 'rare' ? '#FF6B00' : '#007AFF',
          padding: 6,
          display: 'BYCLICK'
        }
      });
    });
    
    this.data.nearbyPlayers.forEach((player) => {
      markers.push({
        id: `player_${player.userId}`,
        latitude: player.latitude,
        longitude: player.longitude,
        iconPath: '/images/map/player.png',
        width: 36,
        height: 36,
        callout: {
          content: player.nickname,
          color: '#FFFFFF',
          fontSize: 12,
          borderRadius: 4,
          bgColor: '#333333',
          padding: 6,
          display: 'BYCLICK'
        }
      });
    });
    
    this.setData({ markers });
  },
  
  initSocket() {
    const socket = require('../../utils/socket');
    this.socket = socket;
    
    socket.on('connect', () => {
      console.log('Socket已连接');
    });
    
    socket.on('player_moved', (data) => {
      this.handlePlayerMoved(data);
    });
    
    socket.on('chest_spawned', (data) => {
      this.loadNearbyElements(this.data.location);
    });
  },
  
  handlePlayerMoved(data) {
    let players = this.data.nearbyPlayers;
    players = players.map(p => {
      if (p.userId === data.userId) {
        return { ...p, latitude: data.latitude, longitude: data.longitude };
      }
      return p;
    });
    this.setData({ nearbyPlayers: players });
    this.updateMarkers();
  },
  
  onMarkerTap(e) {
    const markerId = e.detail.markerId;
    const [type, id] = markerId.split('_');
    
    if (type === 'chest') {
      this.openChest(parseInt(id));
